Reset Journal Entry Odoo App is designed to improve the journal entry management process for users. With this app, users can transition single or multiple journal entries into the draft stage, allowing them to make necessary adjustments and revisions. Additionally, the app simplifies the cancellation of journal entries, enabling users to cancel one or multiple entries with a single click. This functionality not only saves time but also assures accuracy and flexibility in financial record-keeping.

Features

Cancel Multiple Journal Entries

User can cancel single or multiple journal entries by single click.

Reset to Draft Journal Entries

User can reset to draft single or multiple journal entries by single click.
